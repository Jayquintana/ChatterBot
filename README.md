# Chatter Bot

### Summary 
Chatter bot is powered by GPT-3 which is a powerful AI model created by OpenAI. With Chatter Bot you can talk to the AI in free conversations or with questions about the universe!

## Getting Started
1) Clown down this repo 
2) Then ```cd``` into repo
3) Run ```npm install```
4) Run ```npm start```
